How this works: This page is a lightweight, backend-free admin. Changes you save here are stored in this browser's local storage — they'll show up on index.html when you open it on this same browser/device, but visitors on other devices won't see them, since this is a static site with no server or database. To make a change permanent for everyone, use Export JSON below and paste it into the DEFAULT_PROJECTS array in index.html, then re-upload the file wherever it's hosted.
